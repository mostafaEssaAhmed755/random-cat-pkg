# Laravel Random Cat Photo Package

This is a simple Laravel package that provides a random cat photo when you visit the "/random-cat" endpoint in your Laravel application.

## Installation

You can install this package via Composer:

```bash
composer require mostafaessaahmed755/randomcatpkg
```

### Usage

After installing the package, you can immediately use the "`{host}/random-cat`" endpoint in your Laravel application.

Visit the "`{host}/random-cat`" endpoint in your web browser to see a random cat photo.
